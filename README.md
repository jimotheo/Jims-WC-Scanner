# Jims-WC-Scanner
This Chrome extension scans FIFA's official World Cup 2026 resale marketplace and shows you every available seat across all 104 matches — with block, row, seat number, category, and price. It has built-in fair value pricing for every single game so you know instantly whether a ticket is worth buying or overpriced.
When a seat hits fair value or drops below it, you get an instant sound alert so you never miss a deal. Click one button and you're taken directly to that seat on FIFA's site to purchase through their normal checkout.

What's included:





Chrome extension with license key activation



All 104 World Cup 2026 matches



Real-time seat scanning (20-30 seconds per game)



Fair value pricing for every game and category



Instant sound alerts when deals appear



Seat grouping — see consecutive seats together



Search and filter by team, venue, date, or category



One-click navigation to seats on FIFA's site



Step-by-step installation guide for Windows and Mac

How it works:





Install the extension in Chrome



Open FIFA's resale site and log into your account



Click scan on any match



See every available seat with pricing



Get alerted when seats hit fair value



Click through to FIFA and buy

Important notes:





This tool reads publicly available seat data from FIFA's resale marketplace. It does not access your FIFA account, payment info, or personal data.



You must have a FIFA resale account to browse and purchase tickets.



Works on desktop Chrome only (Windows and Mac).



Each license key is for one user.



Scans are limited to one game at a time with a cooldown between scans to ensure reliable performance.

What is fair value?

Fair value is the original face value price that FIFA sold each ticket category for. When resale tickets are listed at or near this price, they represent the best deals available. The scanner tracks fair value for all four ticket categories across every match so you can spot deals the moment they appear.
